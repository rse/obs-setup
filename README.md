
OBS Setup
=========

Version 1.1.31 (2021-11-11)

This is [Dr. Ralf S. Engelschall](https://engelschall.com)'s opinionated
list of software components for installing a full-featured
[Open Broadcaster Software (OBS) Studio](https://obsproject.com/) setup
under Windows 10 (x64). For this, beside the base application, multipleck
[OBS Studio Plugins](https://obsproject.com/forum/resources/categories/obs-studio-plugins.6/)
should be installed.

### Base Application

- **OBS Studio** 27.1.3 (GPL)<br/>
  Open Broadcaster Software (Video Mixing Application)<br/>
  [Homepage](https://obsproject.com/)
  [Download](https://cdn-fastly.obsproject.com/downloads/OBS-Studio-27.1.3-Full-Installer-x64.exe)
